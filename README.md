# 🚗 Parking Lot System (Java)

A simple Parking Lot Management System built using Java and Object-Oriented Programming (OOP) principles.

## 📌 Features

- Park a vehicle
- Remove a vehicle
- Display available parking slots
- Display occupied parking slots
- Check if parking slots are available
- Simple menu-driven implementation

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming
- ArrayList

## 📂 Project Structure

```
ParkingLotSystem/
│── Main.java
│── ParkingLot.java
│── ParkingSlot.java
│── Vehicle.java
```

## 💡 Concepts Used

- Classes and Objects
- Encapsulation
- Object Composition
- Collections (ArrayList)

## ▶️ How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA, Eclipse, or VS Code.
3. Run `Main.java`.
